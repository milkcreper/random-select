ChatGPT:

なるほど、やりたいことを整理すると、こういうマクロですね。

  1. テキストデータの中身を重複なしでランダム抽出

  2. 抽出個数をユーザーが指定できる

  3. 実行ボタン（リンク）をクリックしてマクロを動かせる

  4. 抽出結果を個数の半分で分割表示する

おそらくExcel VBAでの実装が想定されていると思いますが、もしそうであれば以下のようにできます。

VBAマクロ例（Excel）

vba
Sub RandomPickSplit()

    Dim rng As Range
    Dim arr As Variant
    Dim uniqueList As Collection
    Dim countPick As Long
    Dim i As Long, idx As Long
    Dim result() As String
    
    '=== 対象データ範囲を選択 ===
    On Error Resume Next
    Set rng = Application.InputBox("抽出するデータ範囲を選択してください", Type:=8)
    On Error GoTo 0
    If rng Is Nothing Then Exit Sub
    
    '=== 抽出個数の指定 ===
    countPick = Application.InputBox("抽出する個数を入力してください", Type:=1)
    If countPick <= 0 Then Exit Sub
    
    '=== 重複を除去 ===
    Set uniqueList = New Collection
    For i = 1 To rng.Count
        On Error Resume Next
        uniqueList.Add rng.Cells(i).Value, CStr(rng.Cells(i).Value)
        On Error GoTo 0
    Next i
    
    If countPick > uniqueList.Count Then
        MsgBox "指定個数がデータ数を超えています。"
        Exit Sub
    End If
    
    '=== ランダム抽出 ===
    ReDim result(1 To countPick)
    Randomize
    For i = 1 To countPick
        idx = Int(Rnd * uniqueList.Count) + 1
        result(i) = uniqueList(idx)
        uniqueList.Remove idx
    Next i
    
    '=== 結果表示（半分で分割）===
    Dim half As Long
    half = countPick \ 2
    Dim msg As String
    
    msg = "【前半】" & vbCrLf
    For i = 1 To half
        msg = msg & result(i) & vbCrLf
    Next i
    
    msg = msg & vbCrLf & "【後半】" & vbCrLf
    For i = half + 1 To countPick
        msg = msg & result(i) & vbCrLf
    Next i
    
    MsgBox msg, vbInformation, "抽出結果"
    
End Sub

使い方

  1. Excelの開発タブ → Visual Basic → 新しいモジュールに上記コードを貼り付け

  2. Excelに戻り、図形やセルに**ハイパーリンクではなく「マクロの登録」**でRandomPickSplitを割り当てる

  3. リンク（ボタン）をクリックすると、範囲指定→個数入力→結果表示ができます

仕様ポイント

  • 重複除去は Collection を利用

  • 個数を2つのグループに分けて表示（半分ずつ）

  • 個数が奇数の場合、前半が少し少なくなる仕様

  • 「リンクで移動」というより、Excel内で図形ボタンからマクロ起動の形にしています

もし「リンクをクリックするとWeb上で動かす」や「抽出結果をシートに自動書き込み」という仕様にしたい場合は、コードを少し改造できます。

function runRandom() {

使い方（リンクから直接「実行場所へ移動」＆自動実行）

このページは URL パラメータに対応しています：

  • ?count=10 … 抽出個数を10にセット

  • &autorun=1 … ページを開いたら自動で抽出を実行

  • #run … ページ内の実行セクションへスクロール

つまり、
random抽出.html?count=10&autorun=1#run
のようなリンクを踏むと、そのまま実行セクションに飛び、10件で自動抽出されます。
ページ内の 「実行リンクをコピー」ボタン は、いま入力している個数を使ったこの形式のURLを自動生成してコピーします。

データについて

  • いただいた message.txt のリストをページ内に埋め込み済みです。

  • すべて重複なしなので、抽出は常に重複なしになります。

  • データ一覧はページ下部「データ一覧」で確認できます。

よくある質問（Q&A）

  • Q. 個数がデータ数より多いと？
    A. 警告が出て実行されません。まずはデータ数（ページ下部に一覧あり）以下で試してください。

  • Q. 抽出のランダム性は？
    A. 毎回シャッフル（Fisher–Yates）してから先頭を切り出す方式です。

  • Q. 半分に分けるルールは？
    A. floor(個数/2) を前半、残りを後半にしています（奇数は後半が1つ多い）。
